# Environment Variables

Create a `.env.local` file in the project root with the following:

- NEXT_PUBLIC_SUPABASE_URL=
- NEXT_PUBLIC_SUPABASE_ANON_KEY=
- NEXT_PUBLIC_BASE_URL=https://your-domain.com (optional, for production API calls)
- ADMIN_DEFAULT_EMAIL=
- OPENROUTER_API_KEY= (optional, for OpenRouter)
- OLLAMA_URL=http://localhost:11434 (optional, for local Ollama)
- SMTP_URL= (optional, for outbound email)
